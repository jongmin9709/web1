    부모자식과 목록

    밖에 있는 태그를 부모
    안에 있는 태그를 자식
    어떤 태그들은 항상 부모자식관계가 형성되어야 하는것이 있다
    ul,ol과 li태그/ ul은 unordered list로 group을 하기위한 부모고, li가 자식태그가 된다 ol은 ordered list로 자동적으로 순서를 매겨주는 부모역할을 한다.

    구조를 만드는 방법

    1. head = body를 설명하는 태그는 head로 묶기로 약속함
    2. body=본문은 꼭 body라는 태그로 묶어줘야한다
    3. html = 모든것을 묶는 최상위 태그
    4. title =웹페이지의 제목을 설정하는 것 검색엔진이 찾을 때 쓰는것
    5. meta = 파일을 읽는 형식을 정해주는 것 chatset으로 정한다 여기서는 우리가 utf-8로 만들었기 때문에 웹페이지에서도 utf-8로 열어야 읽을 수 있다 또한 이 소스를 보는 사람을에게 파일의 형식을 알려줄 수 있는 하나의 지표가 되는 것
    6. !DOCTYPE = html 본문 맨 위쪽에는 어떤 형식으로 쓰여졌는지를 알려주기 위해 써준다
